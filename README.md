# Car-Price-Prediction-ML
Machine Learning project to predict car prices using Random Forest Regression.
## Project Overview

This project focuses on predicting the price of used cars using Machine Learning algorithms. 
The model analyzes different car features such as brand, year, mileage, fuel type, transmission, 
and other factors to estimate the expected selling price.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook / Google Colab

## Machine Learning Workflow

- Data Collection
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Price Prediction

## Machine Learning Algorithms Used

- Linear Regression
- Random Forest Regression
- Decision Tree Regression

## Dataset

The dataset contains information about used cars including:

- Car Brand
- Year of Manufacturing
- Present Price
- Selling Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission Type
- Owner Details

## Data Analysis Performed

- Handling missing values
- Removing duplicate data
- Data visualization
- Feature correlation analysis
- Identifying important factors affecting car prices

## Model Evaluation

The model performance was evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

## Results

The trained machine learning model successfully predicts car prices based on different vehicle features and provides insights into factors affecting resale value.

## Future Improvements

- Deploy the model using Flask/Streamlit
- Add more real-time car datasets
- Improve prediction accuracy using advanced algorithms

## Conclusion

This project demonstrates how Machine Learning can be used in the automobile industry to estimate fair car prices and support better decision-making for buyers and sellers.
